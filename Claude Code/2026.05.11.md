## 📌 1단계: Claude Code 설치

### macOS / Linux / WSL:

```
curl -fsSL https://claude.ai/install.sh | bash
```

### Windows PowerShell:

```
irm https://claude.ai/install.ps1 | iex
```

### Windows CMD:

```
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

주의/참고:

- `&&` 토큰이 올바른 구문 구분자가 아니라는 오류 → 현재 PowerShell 환경
- `'irm'`을 인식할 수 없다는 오류 → 현재 CMD 환경
- PowerShell에서 프롬프트: `PS C:\`로 표시, CMD에서는 `PS` 없이 `C:\`로만 표시됨
- Windows 환경: Git for Windows 필요, WSL 환경: 불필요
- 백그라운드에서 자동으로 업데이트되어 항상 최신 버전을 유지

### Homebrew

```
brew install --cask claude-code
```

참고:

- `--cask` : 안정 릴리스 채널 추적 및 약 1주일 뒤 업데이트/주요 결함이 있는 릴리스 패스
- `claude-code@latest`로 설치 → 가장 최신의 새 버전 설치
- 자동 업데이트 X
- `brew upgrade claude-code`/`brew upgrade claude-code@latest`로 업데이트 가능

### WinGet

```
winget install Anthropic.ClaudeCode
```

- 자동 업데이트 X
- `winget upgrade Anthropic.ClaudeCode`로 업데이트 가능

## 📌 2단계: 계정 로그인

세션 시작 : `claude` 명령 (계정 필요)

```bash
claude
```

초기 시작 시 로그인 요청

```bash
/login
```

로그인 방법:

- **Claude Pro, Max, Team, 또는 Enterprise** (권장)
- **Claude Console** (선불 크레딧을 통한 API 접근). 첫 로그인 시 중앙 집중식 비용 추적을 위한 "Claude Code" 워크스페이스가 콘솔에 자동 생성
- **Amazon Bedrock, Google Vertex AI, 또는 Microsoft Foundry** (기업 클라우드 공급자)

참고:

- 계정을 전환 시 `/login` 명령 사용

<br>

## 📌 4단계: 첫 번째 질문하기

첫 질문 추천: 코드베이스 이해하기

```
이 프로젝트는 무엇을 하나요?
```

기타 가능한 질문

- 파일 분석
- 요약 제공
- Claude Code의 가능 범위

기타 예시:

- 이 프로젝트에서 사용하는 기술은 무엇인가요?
- 메인 진입점은 어디인가요?
- 폴더 구조를 설명해 주세요
- Claude Code는 무엇을 할 수 있나요?
- Claude Code에서 커스텀 스킬을 어떻게 만드나요?
- Claude Code가 Docker와 함께 작동할 수 있나요?

장점 : 알아서 파일을 찾아 읽기 떄문에 맥락을 지시하지 않아도 됨.

## 📌 필수 명령어

| 명령어               | 기능                                                | 예시                            |
| -------------------- | --------------------------------------------------- | ------------------------------- |
| `claude`             | 대화 세션 시작                                      | `claude`                        |
| `claude "작업"`      | 일회성 작업 실행 후 종료 (수정/실행 등 작업을 수행) | `claude "빌드 오류 수정"`       |
| `claude -p "쿼리"`   | 일회성 쿼리 실행 후 종료 (답변 출력만 존재)         | `claude -p "이 함수 설명해 줘"` |
| `claude -c`          | 현재 디렉토리의 최근 대화 이어가기                  | `claude -c`                     |
| `claude -r`          | 디렉토리와 무관하게 모든 세션 기준 이전 대화 재개   | `claude -r`                     |
| `/clear`             | 대화 내역 지우기                                    | `/clear`                        |
| `/help`              | 사용 가능한 명령어 표시                             | `/help`                         |
| `exit` 또는 `Ctrl+D` | Claude Code 종료                                    | `exit`                          |

## 📌 초보자를 위한 팁

### 1. 요청은 자세히 작성할 것

"버그 고쳐줘" → "잘못된 자격 증명을 입력한 후 사용자에게 빈 화면이 표시되는 로그인 버그를 수정해 주세요"

### 2. 복잡한 작업은 단계를 나누어 지침 작성하기

```
1. 사용자 프로필을 위한 새 데이터베이스 테이블 만들기
2. 사용자 프로필을 가져오고 업데이트하는 API 엔드포인트 만들기
3. 사용자가 자신의 정보를 보고 편집할 수 있는 웹페이지 만들기
```

### 변경 전 Claude가 먼저 코드를 탐색하게 할 것

질문 예시:

- 데이터베이스 스키마를 분석해 주세요
- 영국 고객이 가장 자주 반품하는 제품을 보여주는 대시보드를 만들어 주세요
